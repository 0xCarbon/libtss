# react-native-libtss

High level interface for TSS methods built on top of Rust libtss and React Native module

## Installation

```sh
npm install react-native-libtss
```

## Usage


```js
import { multiply } from 'react-native-libtss';

// ...

const result = await multiply(3, 7);
```
